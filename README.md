# Stock Market Analysis with SQL

![Tech](https://img.shields.io/badge/Technology-MySQL-blue.svg)

## 1. Project Overview
This project analyzes historical stock data for six major companies (Eicher Motors, Bajaj Auto, TCS, TVS Motors, Hero Motocorp, and Infosys). The objective is to use SQL to evaluate price trends, calculate volatility, and generate buy/sell recommendations based on historical performance.

## 2. Technology Used
* **Database:** MySQL
* **Language:** SQL
* **Tools:** MySQL Workbench (Data Import & Analysis)

## 3. Key Insights
* **Highest Growth:** **TVS Motors** showed the highest potential, with a **+267.74%** gain from its lowest to highest price.
* **Strong Performers:** **Infosys (+166.35%)** and **Eicher Motors (+128.95%)** also demonstrated significant volatility and growth.
* **Steady Performers:** **Bajaj Auto (+74.90%)** and **Hero Motocorp (+77.53%)** showed positive but more moderate growth compared to the others.

## 4. Files in this Repository
1.  **`Stock_Analysis_Report.pdf`**: A detailed report summarizing the trend analysis and providing buy/sell recommendations.
2.  **`Assignment_Solutions.sql`**: The complete SQL script used for data cleaning, date formatting, and generating the analysis.
